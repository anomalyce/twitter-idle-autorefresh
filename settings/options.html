General Settings
Look for new tweets every seconds

Twitter's API rate limit is at 180 requests per 15 minutes (12 requests per minute, or every 5th second). It is recommended to not go below 10 seconds as to provide yourself with a bit of a margin.


Whenever the auto-refreshing procedure is enabled, the Twitter logo will be replaced with the Twitter Idle AutoRefresh logo to provide a visual cue.

Please read!

As a privacy proponent myself, I would like to briefly state that this add-on will NEVER record any data whatsoever (e.g. browsing habits). It does however require the storage permission in order to save & read the above user settings. It will also only activate on URLs matching *://twitter.com/* or *://x.com/*.

The add-on is freely available on GitHub for full transparency.
